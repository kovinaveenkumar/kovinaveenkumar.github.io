# Naveen Kumar Kovi — Portfolio

Live site: **https://kovinaveenkumar.github.io**

A fast, responsive, single-file personal portfolio (dark theme) showcasing my projects, skills, publications, and certifications across Data Engineering, Data Science, and ML/AI.

## Run locally
Open `index.html` in any browser, or serve it:

```bash
python -m http.server 8000   # then visit http://localhost:8000
```

## Deploy
Hosted free on GitHub Pages from this repo (`main` / root). Any push to `index.html` auto-redeploys.

## Tech Stack
HTML5 · CSS3 · JavaScript (vanilla)
