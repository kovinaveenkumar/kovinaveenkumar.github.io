## Overview

My personal portfolio — a clean, dark, mobile-friendly site that presents my work to recruiters and collaborators. Built as a single self-contained `index.html` (HTML + CSS + vanilla JS, zero dependencies) so it loads instantly and deploys anywhere.

## Features

- Sticky navigation + smooth scrolling
- Hero with role summary, CTAs, and social links
- Stats band, grouped skills, and project cards linking to each repo
- Experience, education, publication, and certifications sections
- Scroll-reveal animations (IntersectionObserver), fully responsive
- No build step, no frameworks — just open `index.html`

## Run locally

Open `index.html` in any browser (double-click it), or serve it:

```bash
python -m http.server 8000   # then visit http://localhost:8000
```

## Deploy free on GitHub Pages

**Option A — clean URL (recommended):** name the repo **`kovinaveenkumar.github.io`**, push `index.html`, then it's live at `https://kovinaveenkumar.github.io`.

**Option B — this repo:** push to `personal-portfolio-website`, then **Settings → Pages → Branch: main / root → Save**. Live at `https://kovinaveenkumar.github.io/personal-portfolio-website`.
